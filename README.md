## Project Setup

1.  Clone the repository from GitHub:
    ```shell
      git clone https://github.com/dumba23/connect-task.git
      cd connect-task
    ```
2.  Install dependencies:

    ```shell
      npm install
    ```

3.  Copy the `.env` file:
    ```shell
      cp .env.example .env
    ```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
